# booking🎂
基于微信小程序的电影订票系统


#### 介绍🌞

> 系统主要包括用户管理、影院管理、电影管理以及订单管理等模块，详细介绍了用户登录、电影信息展示、座位选择、订单生成等流程的具体实现方法和技术细节。

#### 安装教程🌞

1. 运行环境准备mysql8 + java8 + node14.16.1 + redis

2. 配置maven路径，加载依赖

3. 运行sql文件，确保application.yml的数据库名称和账号密码是数据库所在主机的账号密码
